### 2、数据库的本质

```python
数据库的本质：
	本质其实就是一款基于网络通信的应用程序
    其实每个人都可以开发一款数据库软件 因为它仅仅就是一个基于网络通信的应用程序

    也就意味着数据库软件其实有很多很多
        关系型数据库
            MySQL、oracle、db2、access、sql server
        非关系型数据库
            redis、mongodb、memcache
            
数据库的分类：
    # 关系型
        1 数据之间彼此有关系或者约束
            男生表和前女友表
        2 存储数据的表现形式通常是以表格存储	
            name     password		hobby
            jason	  123			学习
            egon	  123			女教练
            tank	  123			吃生蚝
            每个字段还会有存储类型的限制
            比如姓名只能存字符串...
    # 非关系型
        存储数据通常都是以k,v键值对的形式
```
