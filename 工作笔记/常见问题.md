#### 基本数据类型操作

```python
列表的合并：
	列表中要添加另外一个列表的内容很简单，用加号就可以了
		>>> a = [1,2,3]
        >>> a += [4,5,6]
        >>> a
        [1, 2, 3, 4, 5, 6]
        
 字典的合并：
	字典x 中要添加 另外一个字典y的内容，可以使用字典的update方法
    
        >>> x = {'a':1, 'b': 2}
        >>> y = {'b':10, 'c': 11}
        >>> x.update(y)
        >>> x
        {'a': 1, 'b': 10, 'c': 11}

    从上例中可以发现，如果有重复的key， 会被字典y中的内容取代

    如果我们想将合并后的内容放到一个新的字典对象里面， 而不去修改x，y的内容，该怎么办呢？ 可以使用下面的方法：
    	z = dict(list(x.items()) + list(y.items()))
```
